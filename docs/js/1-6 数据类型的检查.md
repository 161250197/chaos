## 原始类型的判定

六种原始类型可以通过 `typeof` 运算符进行判定：

* undefined - `'undefined'`
* Boolean - `'boolean'`
* Number`'number'`
* String - `'string'`
* BigInt - `'bigint'`
* Symbol - `'symbol'`
